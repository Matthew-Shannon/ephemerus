# Ephemeral
Leverages both ChatGPT and DallE to generate a (text & image) based role playing game.

Player input -> ChatGPT (text) -> DallE (image)


##Install
```bash
git clone https://github.com/Matthew-Shannon/sandbox.git

npm run install

// for dev setup
npm run dev

// for server setup
npm run build
npm run start
```
